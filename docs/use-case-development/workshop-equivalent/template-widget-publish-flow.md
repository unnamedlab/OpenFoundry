# Template, widget, and publish flow

This is the strongest app-composition story currently visible in the repo.

## Sequence

```text
builder
  -> create app or create from template
  -> inspect widget catalog
  -> add and edit pages
  -> preview app
  -> publish app version
  -> expose public or embedded runtime
```

## Why this matters

This is the OpenFoundry-equivalent of a Workshop-style lifecycle: from composition, to preview, to published operational application.
