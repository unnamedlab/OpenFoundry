# Marketplace and packaging

Packaging is where internal tooling becomes distributable product capability.

## Repository signals

`marketplace-service` already contains domain areas for:

- discovery
- registry
- validation
- activation
- dependency handling
- devops-oriented flows

Those signals are visible under `services/marketplace-service/src/domain/*` and `services/marketplace-service/src/handlers/*`.

## Why this matters

This subtree is the natural home for documentation about:

- publishing assets
- promotion gates
- installation and activation
- dependency resolution
- fleet and rollout models
