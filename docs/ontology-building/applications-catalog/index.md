# Applications catalog

Ontology-driven applications are not a single product surface in OpenFoundry. They are a family of user experiences built on the same semantic layer.

## Current candidate families

- object-centric exploration in `apps/web/src/routes/ontology`
- queries and analytical exploration in `apps/web/src/routes/queries`
- maps and geospatial flows in `apps/web/src/routes/geospatial`
- reports in `apps/web/src/routes/reports`
- workflow experiences in `apps/web/src/routes/workflows`
- application composition in `apps/web/src/routes/apps`

## Why this matters

This subtree is where a future Foundry-like application catalog can be described capability by capability.
